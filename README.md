## Built With

- **Engine:** Unity 6000.0.44f1 LTS  
- **Code Editor:** VS Code  
- **Audio Editing:** Audacity  
- **Image Editing:** Photopea  

---

## Game Features

- Smooth bird movement using Rigidbody2D  
- Pipe spawning system  
- Collision detection  
- Score tracking  
- Sound effects (Hit, Win, Game Over)  
- Background music  
- Game pause using `Time.timeScale`
